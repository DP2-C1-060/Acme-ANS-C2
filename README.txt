This is a starter project designed as an educational resource for students enrolled in the Design and Testing
course within the Software Engineering curriculum at the University of Seville. In this version, the project 
focuses on **flight management**, serving as a foundation for the development of information systems related 
to flight administration and operations.

To get this project up and running, please follow the guidelines provided in the theory/lab materials,
ensuring that you link the appropriate version of the Acme-Framework, excluding the following resources:

- **/fragments/**
